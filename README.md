# ShimmerTime v1.1.1

Renamed from DimmerShimmer to ShimmerTime.

Overview
--------
ShimmerTime adds Shimmer-themed emotes, GIFs, and pet icons to your WoW UI and gives you simple settings to enable or disable entire emote categories.

Installation
------------
Place the `ShimmerTime` folder inside your World of Warcraft AddOns directory. 

Example:
- `D:\Blizzard\World of Warcraft\_retail_\Interface\AddOns\ShimmerTime\`
Inside this folder is where you should have your ShimmerTime.toc, Emotes.lua, Quoests.lua files.

What it does
------------
- Provides a set of Shimmer-themed emotes you can show or send to chat.
- Emotes are grouped into categories which can be seen by left clicking the minimap button or visiting the options
- Some emotes include additional resources:
	- Sounds (for example the `bustin` emote).
	- Animated frames (example: the `pascal` GIF with multiple frames and an animation duration).
- Settings allow you to enable or disable entire categories of emotes; the addon ships with the textures, sounds, and frames used by each emote.
- Right click the minimap button or use /shimmer options to see the list of features and emotes!

Slash commands
--------------
- `/shimmer` : Play the Shimmer guild theme locally.
- `/shimmerquote` : Send a random Shimmer quote to the current chat channel.
- `/shimmer options` : Open the addon's options dialog.
- `/shimmer reset` : Reset the minimap button position to its default.
