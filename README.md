# ShimmerTime v1.0.0

Renamed from DimmerShimmer to ShimmerTime.

Included:
- Shimmer category: shimmer
- Shimmer Pets category: cat1, dog1, cat2, dog2, dog3, cat3, cat4, dog4, cat5, cat6, cat7, dog5, cat8
- Dimmer category: lexi, dimmer, cream
- `/shimmer` plays the Shimmer guild theme locally.
- `/shimmerquote` sends a random Shimmer quote to the current chat channel.
- `/shimmer options` opens options.
- `/shimmer reset` resets the minimap button position.
